Chính sách quyền riêng tư — Code Safety Checker

Cập nhật lần cuối: 2026-06-20

Code Safety Checker là extension Chrome (Manifest V3) cho mục đích nghiên cứu khoa học, giúp phát hiện code snippet không an toàn ngay trên trang StackOverflow/GitHub bạn đang xem. Trang này giải thích extension thu thập, sử dụng và lưu trữ dữ liệu gì.

Dữ liệu được gửi đi

Khi extension phát hiện một code block trên trang stackoverflow.com hoặc github.com, nội dung văn bản của code block đó (lấy qua textContent, không bao gồm HTML/script xung quanh) được gửi tới nhà cung cấp LLM bạn đã cấu hình (hiện hỗ trợ Anthropic) để phân tích an toàn. Cùng với code, extension gửi kèm: ngôn ngữ lập trình suy ra từ trang (nếu có) và tên nền tảng (stackoverflow/github).

Extension không gửi: lịch sử duyệt web, cookie, thông tin đăng nhập, địa chỉ IP, hay bất kỳ dữ liệu định danh cá nhân nào khác ngoài nội dung code block.

Dữ liệu được lưu trữ cục bộ

Toàn bộ dữ liệu sau lưu trong chrome.storage.local của trình duyệt bạn — không gửi lên máy chủ nào khác ngoài nhà cung cấp LLM nói trên:

  • API key của nhà cung cấp LLM (nếu bạn tự nhập trong popup) — chỉ lưu cục bộ, không hardcode trong code nguồn.
  • Cache kết quả phân tích, được lưu theo mã băm SHA-256 của nội dung code (không theo URL hay danh tính người dùng), để tránh phân tích lại cùng 1 đoạn code nhiều lần.
  • Số lần gọi API và chi phí ước tính trong ngày, để giới hạn ngân sách (MAX_DAILY_API_CALLS/MAX_DAILY_COST_USD).
  • Trạng thái bật/tắt extension và việc bạn đã đồng ý chính sách này lần đầu hay chưa.

Bên thứ ba nhận dữ liệu

Code snippet được gửi tới Anthropic (nhà cung cấp mô hình Claude) để phân tích. Việc gửi dữ liệu chỉ diễn ra sau khi bạn đã đồng ý với thông báo quyền riêng tư hiển thị trong popup của extension lần đầu cài đặt. Anthropic xử lý dữ liệu theo chính sách quyền riêng tư riêng của họ.

Quyền kiểm soát của bạn

  • Tắt extension bất kỳ lúc nào qua popup — khi tắt, extension ngừng phân tích code mới.
  • Xoá API key đã lưu bất kỳ lúc nào qua popup (để trống rồi bấm Lưu key).
  • Gỡ cài đặt extension qua chrome://extensions sẽ xoá toàn bộ dữ liệu cục bộ liên quan.

Mục đích sử dụng

Dự án này phục vụ mục đích nghiên cứu khoa học (công bố học thuật), không thu thập dữ liệu cho mục đích quảng cáo hay bán cho bên thứ ba.

Liên hệ

Mã nguồn extension và script nghiên cứu được công khai tại repository GitHub của dự án (https://github.com/zstarrynight1/LLMSAFETY). Nếu có câu hỏi về chính sách này, vui lòng mở issue trên repository.
